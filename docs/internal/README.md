# Internal planning docs

These are the working notes Keepou was built from: the macro breakdown into epics
([`EPICS.md`](./EPICS.md)) and the detailed story files under [`stories/`](./stories/),
each with tasks, acceptance criteria and a definition of done.

They record *how* the project was sequenced and are kept for history and context.
They are not needed to run or self-host Keepou — start with the top-level
[`README.md`](../../README.md). For product and technical reference, see
[`../PRD.md`](../PRD.md) and [`../ARCHITECTURE.md`](../ARCHITECTURE.md).
